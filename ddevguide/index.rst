Quick Start
-----------

Here is a list of the basic step necessary to get set up and make a pull request:

1. get :ref:`a clone of DMD and Phobos <setup>` with

      ``git clone https://github.com/D-Programming-Language/dmd.git``

      ``git clone https://github.com/D-Programming-Language/phobos.git``


2. :ref:`build DMD and Phobos <compiling>`.
3. make changes and commit.
4. `send pull request`_.

.. _send pull request: http://help.github.com/send-pull-requests/

Contributing
------------

We encourage everyone to contribute to D and that's why we have put up
this developer's guide. If you still have questions after reviewing the
material in this guide, the :doc:`Developer FAQ <faq>` is another useful
source of information.

Guide for contributing to D:

* :doc:`setup`
* Beginner tasks to become familiar with the development process
    * :doc:`docquality`
* Advanced tasks for once you are comfortable
    * :doc:`fixingissues`
* :doc:`devrole`
* :doc:`communication`
* :doc:`coredev`
    * :doc:`committing`
    * :doc:`devcycle`

It is recommended that the above documents be read in the order listed.
